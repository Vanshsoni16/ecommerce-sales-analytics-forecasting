# 📊 E-Commerce Sales Analytics & Forecasting Platform

## 🚀 Project Overview

The E-Commerce Sales Analytics & Forecasting Platform is an end-to-end Data Science web application built using Python, Pandas, NumPy, Scikit-Learn, Plotly, and Streamlit.

The platform provides an Executive Dashboard for monitoring business performance, analyzing product and regional sales trends, generating actionable insights, and forecasting future sales using Machine Learning.

This project demonstrates the complete Data Science workflow, from data cleaning and analysis to visualization, forecasting, and interactive reporting.

---

## 🎯 Business Objectives

* Analyze historical sales performance
* Monitor key business KPIs
* Identify top-performing products
* Compare regional performance

## 📈 Machine Learning Workflow

### Step 1: Data Preparation

* Load cleaned sales data
* Convert date columns
* Aggregate monthly sales

### Step 2: Feature Engineering

* Create month-based numerical features
* Prepare training dataset

### Step 3: Model Training

* Train Linear Regression model
* Learn historical sales patterns

### Step 4: Forecasting

* Predict future sales
* Generate next 6-month forecast

### Step 5: Evaluation

* Calculate Average Prediction Error
* Calculate Model Accuracy Score (R²)

### Step 6: Business Insights

* Analyze forecast growth
* Generate business recommendations

---

## 📊 Model Evaluation Metrics

The forecasting model is evaluated using:

### Average Prediction Error

Measures the average difference between actual and predicted sales values.

### Model Accuracy Score (R²)

Measures how well the model explains sales trends.
